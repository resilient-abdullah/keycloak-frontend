
# Vue Keycloak Authentication

This project is a **Vue 3 + Vite** frontend that integrates with **Keycloak** for authentication and communicates with a **Laravel API** using secure JWT tokens.

It demonstrates a real-world Single Page Application (SPA) login flow using OpenID Connect.
